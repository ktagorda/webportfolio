About

Hi! I'm Kevin.

At an early age, I was exposed to computers and always dreamed to pursue a career in tech industry specifically in coding. That's why a decided to take a big career leap, resigned from my position and enrolled to a coding bootcamp. On a short span of 2 months, I was able to design and develop a static responsive website using HTML5, CSS3, Bootstrap, and Git for version control, hosted on GitHub, created an E-commerce API with two log in authentication, developed a React-based web application with API integration and responsive design, utilizing modern JavaScript techniques and DOM manipulation.

I've always wanted a career where I can practice my passion. As they say, the more you suppress what you really want, the more it becomes a necessity. Backed with my decade long corporate experience and my passion, I'm more than ready to deep dive and accept challenges in the world of coding.
